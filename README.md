# vTool 房屋投資報酬率計算機

台灣房屋購屋／收租投資試算工具。純網頁單檔 HTML，不用安裝、不用連網，下載後直接用瀏覽器打開就能用。

## 這是什麼

幫你試算「買一間房、貸款、出租幾年後賣掉」整個過程的實際現金流與報酬率：買賣手續費、房貸利息、房地合一稅、租金所得稅都算進去，一次看到不同「貸款成數 × 未來售價」組合下的損益與年化報酬率（IRR）。

## 該用哪個檔案？

| 需求 | 檔案 |
|---|---|
| 只評估**單一個案** | `vTool_v3.html` |
| 同時比較**兩個物件**（例如 A 案新古屋 vs B 案老公寓） | `vTool_compare_v3.html` |
| 其餘 `vTool.html` / `vTool_v2.html` / `vTool_compare.html` / `vTool_compare_v2.html` | 較早的版本，稅制邏輯較簡化，建議直接用 v3 |

## 怎麼用

1. 下載對應的 `.html` 檔案
2. 直接用瀏覽器打開（雙擊檔案，或右鍵「開啟方式」選瀏覽器）
3. 在表單填入：房屋總價、坪數、貸款利率、自訂貸款成數（可多組，用逗號隔開）、貸款年數、寬限期、預期月租金、租金所得個人邊際稅率、預計持有年期、自訂未來售價（可多組）
4. 表格會即時算出每個「貸款成數 × 售價」組合的：
   - 推算總價、賣方仲介費、房地合一稅
   - 累積繳納利息成本
   - 持有期間付出總額（現金流壓力測試用）
   - 實質淨獲利（總額）
   - 實質年化報酬率（IRR）
5. 可以按「匯出 CSV」把整份分析表存下來

## 這個工具算了哪些東西（白話版）

- **買賣成本**：買入固定收 2%（仲介費+代書規費雜支），賣出固定收 4%（仲介費）
- **房貸攤還**：寬限期內只繳利息，寬限期後開始本息平均攤還，逐月模擬到持有期滿或還清為止
- **房地合一稅**：依持有年期自動判定稅率（未滿 2 年 45%、2-5 年 35%、5-10 年 20%、10 年以上 15%）；若自住滿 6 年且沒有出租，會套用 400 萬免稅額 + 10% 優惠稅率
- **租金所得稅**：租金收入以 43% 法定必要費用推計後，併入你設定的個人綜所稅邊際稅率課稅（因為租金所得是併入綜合所得稅，不是單獨稅率，工具沒辦法自動知道你的總所得，需要你自己選一個邊際稅率）
- **實質年化報酬率（IRR）**：不是簡單的「頭尾金額算報酬率」，而是把整個持有期間的每月現金流（租金收入、房貸還款、最終賣屋淨變現）都考慮進去，用現金流折現的方式算出真正貼近實際資金時間成本的年化報酬率

## 已知的簡化（不是 bug）

- 房地合一稅級距的邊界年（剛好持有滿 2／5／10 年整）因為 UI 只能輸入整數年，採用較容易閱讀的判定方式，可能跟法規邊界差一級
- 自住 6 年免稅只檢查「月租金設為 0」，沒有檢查實際戶籍登記、本人+配偶+未成年子女僅一戶等法規要件
- 租金所得稅的 43% 費用推計比例是固定值，無法自訂為實際費用單據

這些都只是估算輔助工具，實際報稅與交易請諮詢地政士／會計師／稅務顧問，不能取代專業意見。
