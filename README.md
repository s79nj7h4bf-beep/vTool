# vTool 房屋投資報酬率計算機

台灣房屋購屋／收租投資試算工具。純網頁單檔 HTML，不用安裝、不用連網，下載後直接用瀏覽器打開就能用。

## 該用哪個檔案？

| 需求 | 檔案 |
|---|---|
| 只評估**單一個案** | `vTool_v3.html` |
| 同時比較**兩個物件**（例如 A 案新古屋 vs B 案老公寓） | `vTool_compare_v3.html` |
| 其餘 `vTool.html` / `vTool_v2.html` / `vTool_compare.html` / `vTool_compare_v2.html` | 較早的版本，建議直接用 v3 |

## 怎麼用

1. 下載對應的 `.html` 檔案，直接用瀏覽器打開
2. 填入房屋總價、坪數、貸款利率／年數／寬限期、預期月租金、租金所得個人邊際稅率、預計持有年期、自訂未來售價
3. 貸款可以選擇輸入**成數(%)** 或**實際貸款金額(萬元)**，表格會即時算出每個「貸款方案 × 售價」組合的成本、稅額、淨獲利與實質年化報酬率（IRR）
4. 可按「匯出 CSV」把整份分析表存下來

這是估算輔助工具，實際報稅與交易請諮詢地政士／會計師／稅務顧問，不能取代專業意見。

## 開發

沒有 build step、沒有 npm 依賴，改完 HTML 存檔就能用。公式細節與開發慣例請參考 [`CLAUDE.md`](./CLAUDE.md)。
